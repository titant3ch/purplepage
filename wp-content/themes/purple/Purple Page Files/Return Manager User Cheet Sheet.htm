Return Manager  User Cheat Sheet

 

Create Order:

  • RMA Number: If the Company Profile is set to have the RMA number “Merchant Assigned”, this will be a required field. If it is set “FedEx Assigned”, it won’t appear.
  • Shipper/Recipient Information: The information on the left is where the Sender Information is to be entered, as in the person returning the package. The information on the right is the Return Location. There’s a dropdown on the bottom of that box that says Another Recipient?. That has a list of all the current Recipient Profiles.
  • Select Shipping Document: This is the type of label you want to process a return for.
  • Label Expiration Date: This is the actual expiration date of the email (not the label). Once the label is printed, it is good for 8 ½ months.
  • Inbound Shipment Details: Fill out everything in Bold. Everything else is optional.

 

Confirmation Screen:

  • This shows all of the details of the shipment. To get the tracking number, click on the Followup Screen button. The tracking number will not be available until after the label is printed.

 

Reports:

  • You can get the status of your Returns for any date range. Click on the number (a hyperlink) to access the information.

 

Search Order:

  • You can search for RMA’s by phone number, confirmation number, and more.

 

Follow up Profile & Follow up Screen:

  • These do not currently work. You can tell the customer they are “depreciated” or “not currently supported”. I have escalated the issue.

 

Add Accounts and Login Profile:

  • These only relate to the FedEx.com profile, not to Return Manager

 

Download:

  • The download file says “.csv”, but it’s not a Comma Separated Values file. It’s a Carat (^) delimited file. Thus, it’s difficult to import into excel. I have escalated the issue.

 

Zip Inquiry:

  • This is only to see the latest pickup for Express available in a given zip code.
