Exporting From WorldShip

 

 

“UPS OnLine Connect” has changed to “IMPORT/EXPORT”

 

 

 

Select the Create / Edit Map.

 

 

 

 

 

 

 

 

 

 

1. Select the Export data from WorldShip Radio button.

 

2. Check the Export .csv box.

 

3. Create a name that you’ll remember – Doesn’t matter what it is.

 

4. Click Create.

 

** Leave Pull-down menu at Address.

 

 

 

 

 

 

 

 

 

 

 

 

 

 

 

 

 

 

 

 

1. Select the ADD ALL COLUMNS because this picture is now wrong

 

2. Click on “OVERWRITE EXISTING FILE”,  then click on “Include Header Row”. This screen now looks different.

 

3. Click OK

 

 

 

 

 

 

 

 

 

 

 

 

 

 

 

 

 

 

 

** Once you have selected a field a chain link will be displayed to the left of the field.

 

 

  1. Select the Drive and folder you want the ( .csv ) file to be exported to.

 

  1. Create a name you’ll remember – Actual name doesn’t matter.

 

  1. Click Save.

 

 

 

 

 

 

 

 

 

Click OK

 

The Map you created will now be displayed in the existing map window.

 

 

 

 

 

 

 

 

 

 

 

 

 

 

 

 

 

 

 

 

 

 

 

 

 

 

 

 

 

Again Select the UPS OnIine Connect.

 

 

This time select the Batch Export option from the pull-down.

 

 

 

 

 

 

 

 

 

 

Select the Map that you have created.

 

Click Next

 

 

 

 

 

 

 

 

 

 

 

 

 

 

 

 

 

 

 

 

 

 

 

The System will display how many records are going to be exported.

 

 

 

 

 

 

 

At this point all you have to do is click on the Next button.  After that a progress bar will be displayed on the screen with a UPS van as the meter.

 

Note:  This export can take up to 30 minutes depending on the actual size of the file.

 

 

 

 

 

 

Once the records have been translated and written the system it will move the data the to .csv file that you requested.  After you click the Save button.

 

Before you quit, find out WHICH VERSION of UPS the customer is running. If the version is 2011 OR LATER, then make sure you select the 2011 import template. Otherwise use the regular UPS import template.

 

That’s it, you’re done.  You have just exported the customer database from the UPS MS Access table.
