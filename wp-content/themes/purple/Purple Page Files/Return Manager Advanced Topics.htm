Multiple Account Numbers:

Click on “Company Profiles”.

 

 

 

 

 

 

 

 

 

 

 

 

 

 

 

Enter each of the account numbers that you want to be able to ship to in the box “Recipient Billing Account Nbr”, one per line. You can do an unlimited number of Express Accounts, but only one SmartPost account.

 

 

 

 

 

 

 

 

 

 

Under “Bill To” Account Number, select the account number that you want to attach to this recipient profile. Note: If you want to ship from multiple account numbers to multiple return locations, use FedEx.com Returns. It will be so much simpler.

 

 

 

 

 

 

 

 

 

 

 

 

 

 

 

 

 

 

 

 

 

 

 

 

 

 

 

 

 

 

 

 

 

 

 

 

SmartPost Return Manager:

Go to SPXREF and enter the account number.

If “Rtn.Elig:” is “N”, you CANNOT DO SMARTPOST RETURNS!!!

If it is “Yes”, you will need to put the Express Account and the SmartPost ID into the Company Profile’s Recipient Billing Account Number, one per line. You will need to enable SmartPost shipping on that screen as well. You will then need to create a SmartPost Recipient Profile and choose the EXPRESS account number from the account number drop down. Then you will have SmartPost specific options in that recipient profile. As we don’t have a smartpost account number to work with, I cannot show you those screen shots.
