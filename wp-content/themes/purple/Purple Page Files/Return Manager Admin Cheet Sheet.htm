Return Manager   Admin Cheat Sheet

 

Company Profile:

  • Recipient Billing Account Number: This is the ONLY PLACE you can add account numbers to Return Manager. Adding accounts anywhere else adds them to the login profile, not Return Manager. Also, each Recipient Profile can only have one account associated.
  • Contact information on the company profile won’t be visible to the customer. The information on the Recipient Profile is what will be shown.
  • Select Return Reason: These options will show up on the Order Screen. You can leave these alone.
  • RMA As Default: This publishes the RMA number to DOTCOM and puts it on the label so you have visibility in reports and Insight.
  • RMA Assignment Flag: This determines if FedEx assigns the RMA or the customer. The RMA Auto Start Number is only relevant if FedEx assigns the RMA number.
  • Default Shipping Document: What type of label you want to be default.
  • Express Shipping Document: Return Manager doesn’t print labels. Air bill is if you’re going to manually fill out an Air Waybill. Express Tag is a label where FedEx will deliver the label for a charge to the customer and pick up the package. Online Label is an email label. More information is in our FedEx Service Guide: http://images.fedex.com/us/services/pdf/Service_Guide_2012.pdf
  • Ground Shipping Document: The Online Label is an email label. The Ground Call Tag is a label where FedEx delivers the label to the customer for a charge and then picks up the package.
  • Smart Post Shipping Document: Get with Mark Collins. This can get complicated.
  • Return Shipper Method: Leave as is, it is also on the Order Screen.
  • FUP Flag: This is in regards to the Follow up Feature. This feature isn’t working. Until it’s fixed, you can tell a customer its “depreciated” meaning no longer supported.
  • Email Confirmation: This will send a confirmation to the sender in addition to any email label sent.

 

Label Profile

  • Maximum number of days for label printing: This will set the max number of days for label printing, but they will still need to specify the ACTUAL number of days allowed for label printing on the order screen.
  • Merchant Default Email: Putting an email there will send a copy of the label to that email.
  • Email Confirmation: This will send a confirmation email to the sender.

 

Recipient Profile

  • By default, this screen is blank. This is because there is already a default Recipient Profile. To get to it, press List at the bottom of the screen.
  • If you need to work with multiple Recipient Profiles or multiple Account Numbers please get with Mark Collins.

 

Manage Users

  • You can invite new users and change the administrator. There can be only one Administrator.
