Return Manager  Probing Cheat Sheet

 

 

 

 

What Type of Returns do they Need?

 

  • Return Manager: Email Labels and Call Tags Only
  • FedEx.com Returns: Print labels, Email Labels, and Call Tags
  • Return Stamps: Their Sales Rep can order pre-printed return labels

 

Do They Use Smart post?

 

  • Are they using SmartPost for Returns?
  • If so, are they enabled for SmartPost Returns? This is different than normal SmartPost. KB 13568. You’re looking for Return Enablement.
  • If the SmartPost Enabled flag is Y but Rtn.Elig is N, close the ticket and ask Sales for a new ticket with the right account number.
  • Smartpost accounts can be set for shipping OR returns, not both. You need separate account numbers to do both.
